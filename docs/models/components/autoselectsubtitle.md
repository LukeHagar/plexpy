# AutoSelectSubtitle

The auto-select subtitle mode (0 = Manually selected, 1 = Shown with foreign audio, 2 = Always enabled)


## Values

| Name                       | Value                      |
| -------------------------- | -------------------------- |
| `MANUALLY_SELECTED`        | 0                          |
| `SHOWN_WITH_FOREIGN_AUDIO` | 1                          |
| `ALWAYS_ENABLED`           | 2                          |
