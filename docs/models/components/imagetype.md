# ImageType

Describes both the purpose and intended presentation of the image.


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `BACKGROUND`        | background          |
| `BACKGROUND_SQUARE` | backgroundSquare    |
| `BANNER`            | banner              |
| `CLEAR_LOGO`        | clearLogo           |
| `COVER_POSTER`      | coverPoster         |
| `SNAPSHOT`          | snapshot            |
