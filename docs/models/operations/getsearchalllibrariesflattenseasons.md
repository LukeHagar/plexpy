# GetSearchAllLibrariesFlattenSeasons

Setting that indicates if seasons are set to hidden for the show. (-1 = Library default, 0 = Hide, 1 = Show).


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `LIBRARY_DEFAULT` | -1                |
| `HIDE`            | 0                 |
| `SHOW`            | 1                 |
