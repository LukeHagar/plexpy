# GetLibrarySectionsAllQueryParamType

The type of media to retrieve or filter by.
1 = movie
2 = show
3 = season
4 = episode
E.g. A movie library will not return anything with type 3 as there are no seasons for movie libraries



## Values

| Name          | Value         |
| ------------- | ------------- |
| `MOVIE`       | 1             |
| `TV_SHOW`     | 2             |
| `SEASON`      | 3             |
| `EPISODE`     | 4             |
| `ARTIST`      | 5             |
| `ALBUM`       | 6             |
| `TRACK`       | 7             |
| `PHOTO_ALBUM` | 8             |
| `PHOTO`       | 9             |
