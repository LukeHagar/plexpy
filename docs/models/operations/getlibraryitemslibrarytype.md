# GetLibraryItemsLibraryType

The type of media content in the Plex library. This can represent videos, music, or photos.



## Values

| Name          | Value         |
| ------------- | ------------- |
| `MOVIE`       | movie         |
| `TV_SHOW`     | show          |
| `SEASON`      | season        |
| `EPISODE`     | episode       |
| `ARTIST`      | artist        |
| `ALBUM`       | album         |
| `TRACK`       | track         |
| `PHOTO_ALBUM` | photoalbum    |
| `PHOTO`       | photo         |
| `COLLECTION`  | collection    |
