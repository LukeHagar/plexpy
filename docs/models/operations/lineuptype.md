# LineupType

- `-1`: N/A
- `0`: Over the air
- `1`: Cable
- `2`: Satellite
- `3`: IPTV
- `4`: Virtual



## Values

| Name      | Value     |
| --------- | --------- |
| `MINUS_1` | -1        |
| `ZERO`    | 0         |
| `ONE`     | 1         |
| `TWO`     | 2         |
| `THREE`   | 3         |
| `FOUR`    | 4         |
